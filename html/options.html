List the websites you want to block, one per line. For example:

cnn.com
nytimes.com
google.com/news
//x.com
www.chess.com

To block all sites except a select few, enter * on the first line and prefix allowed sites with +:

*
+stackoverflow.com
+msdn.microsoft.com
